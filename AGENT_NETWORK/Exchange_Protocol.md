# Protocole d'Échange Bitcoin ⇄ Cash

## Règles de Base
- Taux de change : Fixé quotidiennement basé sur le marché local.
- Commission : Max 2% par transaction.

## Processus pour l'Agent
1. Vérifier l'identité du client (pièce d'identité).
2. Pour vendre du Bitcoin :
   - Le client envoie le Bitcoin à l'adresse de l'agent.
   - L'agent verse le cash immédiatement.
3. Pour acheter du Bitcoin :
   - Le client donne le cash.
   - L'agent envoie le Bitcoin immédiatement.

## Sécurité
- Ne jamais garder de grandes sommes en cash.
- Utiliser un portefeuille Lightning pour les petites transactions.

---

*Ce protocole est open source et peut être adapté.*
